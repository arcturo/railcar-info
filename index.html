Get started quickly

Get running on Rails no compiler required (though you can install from source if you'd like). From the start, Railcar will bootstrap you with all the essentials to get started with Rails with minimal hassle. Simply click a button and it's all installed to the application bundle. Done with Rails or want to move to another stack? Simply delete the application bundle and it all disappears.

You can also install other, non-Ruby packages like databases and languages. For example, you might be curious about NoSQL databases like Redis or Clojure. Install them with the package manager (powered by a sequestered installation of homebrew which won't bother your system-wide install should you have one) and have them up and running in no time!

Railcar's streamlined UI makes it simple to learn about and install Ruby versions. As you learn more about Rails and Ruby, you might want to check out alternative interpreters like Rubinius or integrate with some of your existing Java code with JRuby. No problem. With one click, you can build or install one of these interpreters and you can immediately start using it with your Rails applications.

Get Real World™ tools

While Railcar's UI helps you through the tough and confusing parts of Rails and web development, under the hood Railcar uses the real tools that real Rails developers use everyday. Once you're comfortable, you can drop straight to the command-line and start using them like normal. Using tools like homebrew, RbEnv, and others will make contributing to OSS and your team at work that much easier!

Transition with ease

Need a more custom setup? Join a team with a strictly defined workflow? No problem. You can easily transfer your Railcar environment to a system-wide install with a single click. Railcar will place your homebrew packages in the proper place and install your Ruby to a system install of RbEnv or (when the rvm mount feature is implemented) RVM.
